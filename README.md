# realfakeManager
使用React、Redux、Redux-Saga、Axios等作为前端逻辑开发
使用Antd作为前端UI开发
使用React-Loadable分割代码、按需加载
使用Webpack打包工具打包工程
游客和用户访问地址：https://www.realfake.cn
管理员后端地址：https://github.com/smalldragon1997/realfakeManager
